# NEXUS | Interactive 3D Component Hub

Welcome to **Nexus**, an interactive standard for high-performance 3D web components. This standalone repository serves as a dark-mode, futuristic hub featuring cutting-edge React components integrated with WebGL and Spline 3D.

## Features

- **Cinematic Boot Sequence**: A highly immersive, glitch-art loader to build anticipation.
- **Dynamic Asset Scanner**: "Wireframe overlay" styled 3D model loaders that feel like an operating system interface.
- **Responsive Accordion UI**: A custom, Framer Motion-powered horizontal expanding gallery that fluidly adapts to both desktop and mobile layouts.
- **Mobile-First UX**: Includes two-tap interactions for touch interfaces, ensuring no instant accidental scene launches.
- **Advanced 3D Integrations**:
  - `ai-robot`: Interactive humanoid model.
  - `scene-2`: Quantum routing core concepts.
  - `scene-3`: Typographic glass refraction studies.
  - `scene-4`: 3D Nepal Cricket Jersey interaction.

## Technologies Used

- **Next.js 15 (App Router)**
- **React & TypeScript**
- **Framer Motion** (for complex fluid layout transitions and animations)
- **Tailwind CSS** (for styling and responsive design)
- **Spline 3D** (`@splinetool/react-spline`)

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the hub in action.

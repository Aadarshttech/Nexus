## 🌌 Overview

**Nexus** is a state-of-the-art interactive hub designed to showcase high-fidelity 3D web components. Built with a focus on immersive aesthetics and fluid performance, Nexus bridges the gap between traditional UI and futuristic 3D environments.

> "A gateway to the next generation of interactive web experiences."

---

## ✨ Key Features

### 🚀 Boot Sequence
Experience a cinematic entry with a glitch-art loading sequence, system status readouts, and a dynamic progress visualizer that sets the tone for the journey ahead.

### 🔍 Wireframe Scan Loader
Every 3D asset is loaded behind a sophisticated "Wireframe Scanner" UI, providing a high-tech "acquiring asset" feel that ensures a premium user experience from start to finish.

### 📱 Fluid Responsive Gallery
A custom-built horizontal accordion gallery that adapts intelligently:
- **Desktop**: Expansive, hover-responsive layout with vertical typography.
- **Mobile**: Stacking vertical layout with tap-to-expand logic and rounded "glassmorphism" containers.

### 🎮 Interactive 3D Modules
Explore a curated selection of 3D experiments:
- **Humanoid AI**: Interactive robot with system HUD overlays.
- **Neural Core**: Quantum processing visualization.
- **Glass Typo**: Studies in real-time refraction and typography.
- **Sports Apparel**: High-detail 3D jersey visualization for the Nepal National Cricket Team.

## 🛠 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **3D Engine**: [Spline Runtime](https://spline.design/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Type**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Aadarshttech/Nexus.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to enter the Nexus.
